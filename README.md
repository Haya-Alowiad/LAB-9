Which model performed better, Decision Tree or Random Forest? Explain your answer.

The Decision Tree performed better than the Random Forest model.

Although Random Forest models typically achieve higher accuracy, in this case the model was affected by class imbalance and showed very poor recall for class 1. The confusion matrix indicates that it failed to correctly classify most instances of class 1.

In contrast, the Decision Tree provided a more balanced performance and was better at identifying class 1, making it the more suitable model for this dataset.
